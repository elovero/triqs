StateVector ConceptÂ¶

  • PurposeA container type, which is semantically an element of a vector space over some number field.

    The vector space must be equipped with a scalar product. Elements of the container are idenified with expansion coefficients with respect to some fixed basis. The container provides random read/write access to its individual elements as well as read-only iteration using foreach function.
