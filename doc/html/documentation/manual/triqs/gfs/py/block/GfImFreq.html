[GfImFreq] Matsubara Green’s function in imaginary frequencies¶

Reference¶

Warning

Arguments of __init__() must be documented.

HDF5 data scheme¶

The GfImFreq (TRIQS_HDF5_data_scheme= “GfImFreq”) is decomposed in the following objects:

Name

Type

Meaning

Mesh

MeshGf

The mesh

Data

3d numpy of complex
Data[n,i1,i2] is the element of the Green function where:

  • n is the Matsubara index

  • i1, i2 are the indices

For real GF in time, we store only starting from n=0 and the rest is deduced by symmetry

IndicesL,IndicesR

string

The Python repr of the indices, e.g. (1,2), or (1,) repr(this_string) reproduces the indices

Name

string

Name of the Green function block

Note

string

Note

Example¶

from pytriqs.gf import GfImFreq, SemiCircular

g = GfImFreq(indices = ['eg1','eg2'], beta = 50, n_points = 1000, name = "egBlock")

g['eg1','eg1'] = SemiCircular(half_bandwidth = 1)
g['eg2','eg2'] = SemiCircular(half_bandwidth = 2)

from pytriqs.plot.mpl_interface import oplot, plt
oplot(g, '-o', x_window = (0,10))
plt.ylim(-2,1)
