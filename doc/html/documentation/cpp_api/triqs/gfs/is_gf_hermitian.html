triqs::gfs::is_gf_hermitian¶

#include <triqs/gfs.hpp>

Synopsis

template<typename G>
bool is_gf_hermitian (G const & g, double tolerance = 1.e-13)

Test if a Green function object fullfills the fundamental property G[iw](i,j) = conj(G[-iw](i,j)) up to a tolerance \(\epsilon\)

Template parameters¶

  • The Green function type

Parameters¶

  • g The Green function object to check the symmetry for

  • tolerance The tolerance \(\epsilon\) for the check

Returns¶

true iif

\[\forall n,\; \max_{ab}|g^*_{ba}(-i\omega_n)-g_{ab}(i\omega_n)|<\epsilon\]
